🎤 面試口條建議

Array in-place 題：說這和「搬移 memory buffer」一樣，避免額外 RAM allocation。

Bit 題：說這就是「操作硬體 register flag」的對應題。

Queue/Stack 題：說這和「UART TX FIFO / ISR producer-consumer」一模一樣。

Sliding window 題：可以聯想到「限流 / debounce / timer tick」。
